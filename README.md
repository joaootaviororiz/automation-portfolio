# 🚀 Portfólio de Automação com Python

Este repositório reúne projetos de automação de processos utilizando Python, com foco em ETL, integração de dados e geração de insights para áreas de negócio.

## 📊 Projetos

### 🔹 ETL Firebird → Google Sheets
Pipeline de dados automatizado que extrai informações de banco Firebird, transforma os dados e envia para o Google Sheets para análise e visualização.

📁 Caminho: `automation-portfolio/`

## 🛠️ Tecnologias utilizadas

- Python
- SQL
- Firebird
- Google Sheets API
- Pandas
- Automação com scripts
